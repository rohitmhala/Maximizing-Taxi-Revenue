# 🚕 Taxi Fare Analysis – Payment Type vs Revenue

## 📌 Project Overview

This project explores how different **payment methods** (Cash vs Card) affect **fare amounts** in the taxi booking sector. Our goal is to provide **data-driven insights** to help drivers and platforms **maximize revenue**.

---

## ❓ Research Question

**Does the type of payment method impact the fare amount paid by customers?**

---

## 📊 Dataset

Sample fields:
- `passenger_count`
- `trip_distance`
- `payment_type`
- `fare_amount`
- `duration`

![Sample Data](./Screenshot%202025-08-01%20221600.png)

---

## 🔍 Analysis Summary

- **Card payments** are preferred by 78% of customers.
- **Card fares** tend to be higher and for longer trips.
- **Statistical tests** confirm a significant relationship between payment type and fare amount.
